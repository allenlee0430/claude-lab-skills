# Filing scheme

| Folder | Scope |
|---|---|
| `00_Active_Now` | Current deliverables requiring attention |
| `10_Research_Manuscripts` | Papers, drafts, reviews, SI, cover letters |
| `20_Grants_Awards` | Grants, fellowships, awards, budgets, support letters |
| `30_Presentations` | Slide decks |
| `40_Data_Protocols_Images` | Data, protocols, images, spreadsheets, raw exports |
| `50_Lab_Admin_Procurement` | Invoices, POs, receipts, contracts, HR and facility admin |
| `60_Personal_Legal_Finance` | Passports, visas, tax, banking, insurance, medical and IDs |
| `70_Code_Automation` | Scripts, notebooks, LaTeX and structured code assets |
| `80_Software_Installers` | Application installers |
| `90_Archive_To_Review` | Ambiguous or low-confidence material |

Apply the same scheme to Desktop and Downloads. Keep active work on Desktop small; Downloads is an intake queue, not a permanent archive.
