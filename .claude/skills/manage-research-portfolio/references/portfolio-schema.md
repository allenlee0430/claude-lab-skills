# Portfolio schema

Core fields: project ID, project name, domain, outcome, stage, status, owner, collaborators, next action, next decision, hard deadline, internal target, dependency, risk, confidence, last evidence date, source link, and notes.

Suggested domains: manuscripts, grants, experimental programs, trainees, talks and teaching, lab operations, commercial or partnership work, and personal administration.

Status vocabulary: active, waiting, blocked, scheduled, submitted, accepted, paused, archived. Keep status factual; express concern in risk and confidence fields.
