# Revise manuscripts

1. Identify the authoritative manuscript version, supporting information, figures, reviewer letter, and journal instructions. Do not mix stale drafts.
2. Build a claim-to-evidence map before substantive rewriting. Flag claims not supported by a figure, table, source, or cited paper.
3. Preserve scientific meaning, numerical values, nomenclature, citation anchors, and tracked reviewer commitments.
4. For rebuttals, structure each item as reviewer comment, response, action taken, exact revised text, and manuscript location. Distinguish completed changes from respectful disagreement.
5. Check abbreviation definitions, ionizable/cationic terminology, gene and protein nomenclature, sample sizes, statistics, figure order, data availability, and author-contribution consistency.
6. Keep prose restrained, hypothesis-led, and specific. Apply the available `humanizer` skill only after the technical pass.
7. Deliver the revised artifact plus a concise submission checklist and unresolved scientific questions.

Read `references/editorial-standard.md` before a final submission pass.
