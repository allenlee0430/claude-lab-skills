# Draft research grants

1. Read the call, instructions, scoring rubric, deadline, and current template before drafting.
2. Search `~/Desktop/20_Grants_Awards`, `~/Downloads/20_Grants_Awards`, and `~/Desktop/00_Active_Now` for prior language and evidence. Prefer funded or final materials over early drafts.
3. Build a requirement matrix: section, limit, scoring criterion, required evidence, source file, and owner.
4. Draft to the scoring criteria. Lead each aim with gap, hypothesis, intervention, readout, success threshold, risk, and alternative.
5. Separate established results, preliminary data, planned work, and aspiration. Never invent citations, numbers, collaborations, equipment, or commitments.
6. Produce both technical and plain-language summaries when the mechanism permits.
7. Run scientific consistency, budget-to-aim, page-limit, attachment, and institutional-signature checks.
8. Apply the available `humanizer` skill to final narrative prose while preserving technical precision.

Use the document skill for `.docx`, the spreadsheet skill for budgets, and the PDF skill for call documents. Read `references/program-context.md` for recurring research framing.
